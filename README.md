# 活力健身馆网站

一个功能完整的健身馆管理系统，包含训练推荐、AI助手和管理后台三大核心功能。

## 📁 文件结构

```
fitness-gym/
├── index.html          # 主页（导航页）
├── training.html       # 训练推荐页面
├── training.js         # 训练推荐功能逻辑
├── assistant.html      # AI助手页面
├── assistant.js        # AI助手功能逻辑
├── admin.html          # 管理后台页面
├── admin.js            # 管理后台功能逻辑
├── styles.css          # 公共样式文件
└── README.md           # 项目说明文档
```

## 🎯 功能模块

### 1. 训练推荐 (training.html)
- 选择需要锻炼的身体部位（手臂、下肢、核心、胸部、背部、心肺）
- 支持多选组合
- 智能推荐训练课程和动作
- 显示训练时长和频率建议

### 2. AI助手 (assistant.html)
- 集成Claude AI的智能对话系统
- 提供健身建议和营养指导
- 路线导航（地铁/开车）
- 快捷问题按钮
- 实时对话功能

### 3. 管理后台 (admin.html)
- 教练管理：查看工作天数、自动计算工资
- 会员管理：记录喜好、训练频率、备注
- 数据统计：总数、工作天数、应付工资
- 完整的增删改功能

## 🚀 使用方法

### 基础使用
1. 直接用浏览器打开 `index.html` 即可开始使用
2. 点击导航按钮进入不同功能模块
3. 每个模块都可以独立使用和调试

### AI助手配置
AI助手功能需要配置Anthropic API。默认情况下，代码已经集成了API调用逻辑，可以直接使用。

**注意**：Claude API在claude.ai环境中无需API密钥即可使用。如果在本地环境使用，需要：
1. 注册Anthropic账号获取API密钥
2. 在 `assistant.js` 中添加API密钥到请求头

## 🔧 调试指南

### 训练推荐模块调试
1. 打开 `training.html`
2. 修改 `training.js` 中的 `trainingPrograms` 对象来自定义训练课程
3. 调整 `toggleBodyPart()` 函数来改变选择逻辑

### AI助手模块调试
1. 打开 `assistant.html`
2. 在 `assistant.js` 中修改系统提示词（system参数）
3. 调整 `sendMessage()` 函数来改变对话逻辑
4. 可以在浏览器控制台查看API调用日志

### 管理后台模块调试
1. 打开 `admin.html`
2. 修改 `admin.js` 开头的 `coaches` 和 `members` 数组来设置初始数据
3. 调整 `calculateSalary()` 函数来改变工资计算方式
4. 所有数据存储在内存中，刷新页面会重置

## 💡 扩展建议

### 数据持久化
目前数据存储在内存中。如需永久保存，可以：
1. 使用 `localStorage` 保存在浏览器
2. 连接后端数据库（MySQL、MongoDB等）
3. 使用云数据库服务

### 增强功能
- 添加会员登录系统
- 课程预约功能
- 在线支付集成
- 数据导出（Excel、PDF）
- 图表统计（教练业绩、会员趋势）

### 样式定制
所有样式都在独立的 `<style>` 标签或 `styles.css` 中，可以轻松修改：
- 颜色方案：修改渐变色值
- 字体大小：调整各元素的 `font-size`
- 布局间距：修改 `padding`、`margin`、`gap`

## 🆘 常见问题

### AI助手无法回复
- 检查网络连接
- 确认API调用是否成功（查看浏览器控制台）
- 在claude.ai环境下可直接使用，无需配置

### 数据刷新后丢失
- 这是正常现象，数据存储在内存中
- 需要持久化请参考"扩展建议"部分

### 页面样式错乱
- 确保所有文件在同一目录下
- 检查 `styles.css` 是否正确加载
- 清除浏览器缓存后重试
